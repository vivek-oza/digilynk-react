# DIGILYNK

## Installation and Setup

### Clone the Repository
```sh
git clone github.com/vivek-oza/digilynk-react
cd frontend
```

### Install Dependencies
Ensure you have **Node.js** and **npm** (or **yarn**) installed. Then, run:
```sh
npm install
# or
yarn install
```

### Run the Project
Start the development server with:
```sh
npm run dev
# or
yarn dev
```
